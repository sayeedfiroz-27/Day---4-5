# Python Day 4 and Day 5 Notes

Detailed classroom notes in simple Hinglish.

## Files

- `index.html` - Browser page with sidebar navigation for Day 4 and Day 5.
- `day4_student_marks_management_system.md` - Mini project notes for Student Marks Management System.
- `day4_expanded_practice.md` - Extra Day 4 practice bank with 2-3 examples for each project feature.
- `day5_functions_strings_data_structures.md` - Functions, strings, and data structures notes.
- `day5_expanded_practice.md` - Extra Day 5 practice bank with additional examples and detailed explanations.
- `examples/day4_student_marks_system.py` - Runnable Day 4 project.
- `examples/day5_practice.py` - Runnable Day 5 practice examples.

## How to Use

Open `index.html` in a browser and switch between Day 4 and Day 5 using the sidebar.

Run examples:

```bash
python examples/day4_student_marks_system.py
python examples/day5_practice.py
```
